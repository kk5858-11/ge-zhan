# Prisma Landing Page

这是一个基于 `Vite + React + TypeScript + Tailwind CSS` 的个站落地页项目，主题风格为暗黑电影感（dark/moody/cinematic）。

## 技术栈

- React 18
- TypeScript
- Vite 7
- Tailwind CSS 3
- framer-motion（动画）
- lucide-react（图标）

## 页面结构

- Hero（全屏视频 + 悬挂导航 + 动态标题）
- About（多字体拉升动画 + 滚动字符显隐）
- Features（噪声背景 + 卡片入场动画）

## 本地开发

```bash
npm install
npm run dev
```

默认地址：

`http://localhost:5173`

## 生产构建

```bash
npm run build
npm run preview
```
